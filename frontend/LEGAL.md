## Verify / Consider

- [ ] **Form an LLC** - ~$100/year in Oregon, separates personal assets from project liability

- [ ] **Data security audit** - Verify:
  - Passwords are hashed (not stored in plain text)
  - HTTPS everywhere
  - Database access is restricted
  - No sensitive data in logs
  - Backup strategy exists

- [ ] **GDPR compliance** (if EU users possible) - Right to:
  - Access their data
  - Delete their data
  - Export their data
  - Know what's collected

- [ ] **CCPA compliance** (California users) - Similar to GDPR, disclose data practices

- [ ] **Email compliance** - CAN-SPAM requires:
  - Clear sender identification
  - Unsubscribe option in marketing emails
  - Honor opt-outs promptly

- [ ] **Copyright for book covers** - Verify fair use for displaying cover images from Open Library / Google Books API

- [ ] **Consult a lawyer** - One-time consultation for peace of mind, especially regarding:
  - Liability waivers in ToS
  - Privacy policy completeness
  - Any Oregon-specific requirements
